# OpenDocs

**Open-source, client-side PDF & document management.**

OpenDocs lets you view, organize, merge, and edit PDFs and documents entirely in your browser — no server uploads, no accounts, your files never leave your device.

## Features

- 📄 **PDF Viewing & Editing** — Open, view, and annotate PDFs client-side
- 🔀 **Merge & Split** — Combine multiple PDFs or extract pages
- 🗂️ **Document Organization** — Drag-and-drop page reordering
- 🔒 **Privacy First** — All processing happens in-browser; files are never uploaded
- 📱 **Responsive** — Works on desktop, tablet, and mobile

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [pdf-lib](https://pdf-lib.js.org/) — PDF manipulation
- [pdfjs-dist](https://mozilla.github.io/pdf.js/) — PDF rendering
- [Zustand](https://zustand-demo.pmnd.rs/) — State management
- [Tailwind CSS](https://tailwindcss.com/) — Styling

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
